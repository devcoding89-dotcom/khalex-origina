# **App Name**: GameZone Marketplace

## Core Features:

- Customer Product Catalog: Customers can browse a wide array of gaming phones, laptops, gadgets, Call of Duty accounts, and CP top-up services, with integrated filters and search functionality.
- Shopping Cart & Checkout: Allows customers to add multiple items to a shopping cart and proceed to a simplified checkout, capturing essential contact information for manual order fulfillment via WhatsApp.
- Secure Admin Login: A protected administrative login page (username/password) to control access to the store's backend management features.
- Product & Service Management: Admin users can easily add new items, edit existing products (physical like phones/laptops/gadgets, and digital like CoD accounts), and services (CP top-ups), including details, images, pricing, and stock levels.
- Order Management Dashboard: Provides administrators with a clear overview of all customer orders, enabling them to track order status, update fulfillment progress, and communicate directly with customers for payment and delivery arrangements.
- AI-Powered Product Description Tool: Integrate a generative AI tool within the admin panel to assist with drafting engaging and optimized product descriptions or marketing copy for newly added items or services.
- Client-Side Data Persistence: Utilize browser-based LocalStorage for efficiently managing and persisting all product, order, and admin data locally for this MVP.

## Style Guidelines:

- Color scheme: Dark theme, embodying a modern and immersive gaming aesthetic.
- Primary color: A dynamic electric blue (#7AC7FF) used for headers, active states, and primary navigational elements, ensuring a high-tech feel.
- Background color: A deep, subtly desaturated blue (#14191F) to create a premium, immersive backdrop for content.
- Accent color: A sharp neon green (#17E517) applied to call-to-action buttons, interactive highlights, and critical alerts to command attention and enhance visibility.
- Headlines and prominent display text: 'Orbitron' (sans-serif), chosen for its futuristic and bold character, perfectly suiting the gaming theme.
- Body text and general content: 'Rajdhani' (sans-serif), providing excellent readability and a contemporary, clean look across the application.
- Deployment of Font Awesome icons, prioritizing symbols and glyphs with a strong connection to gaming, technology, and commerce, consistent with the store's offerings.
- Employ a structured grid system for displaying products, enabling clear organization and an intuitive browsing experience. Admin panels will feature logical forms and data tables for efficient management workflows.
- Subtle and quick transitions on hover states, clicks, and page loads, designed to add a layer of interactivity and polish without detracting from functionality or causing delays.
